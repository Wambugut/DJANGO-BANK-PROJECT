# Django Bank Management System

A banking management system developed using Django (Python) to simulate core banking operations, including account creation, transactions, and balance tracking. This project demonstrates backend development skills, database management, and secure transaction logic.

## Features
- User registration and authentication
- Create and manage bank accounts
- Perform deposits, withdrawals, and transfers
- Transaction history tracking
- Admin dashboard for account management

## Technologies Used
- Python (Django Framework)
- SQLite / PostgreSQL
- HTML, CSS, JavaScript (for basic frontend templates)
- Git & GitHub for version control

## Purpose
The purpose of this project is to simulate real-world banking operations and showcase backend system design, database handling, and secure application logic. It provides hands-on experience in building a full-stack application with a strong focus on backend functionality.

## Installation (Optional)
1. Clone the repository:  
   `git clone https://github.com/Wambugut/django-bank-project.git`
2. Navigate to the project folder:  
   `cd django-bank-project`
3. Install dependencies:  
   `pip install -r requirements.txt`
4. Run migrations:  
   `python manage.py migrate`
5. Start the server:  
   `python manage.py runserver`
6. Access the app at `http://127.0.0.1:8000/`
